- **Color Scheme:** Primary blue (\#0066CC), secondary green (\#00AA33), neutral grays (\#F5F5F5, \#666666).

## Design Principles for All Mockups

1. **Clarity \& Readability:**
    - Font: Sans-serif (Roboto, Open Sans) at 16px base.
    - Line height: 1.5x for comfortable reading.
    - Color contrast: WCAG AA 4.5:1 minimum.
2. **Multi-Lingual Layout:**
    - Hebrew (RTL) and English (LTR) mirror each other.
    - UI elements adapt direction (icons left/right, alignment).
3. **Mobile Responsiveness:**
    - Stack components vertically on screens < 768px.
    - Collapsible sidebars and drawers on mobile.
    - Touch-friendly targets (min 48x48px).
4. **Accessibility:**
    - Keyboard navigation support.
    - ARIA labels and semantic HTML.
    - High-contrast mode option.
    - Screen reader-friendly.
5. **Gamification Integration:**
    - Points and badges visible but not intrusive.
    - Optional leaderboards (user can hide).
    - Celebration animations on achievement.
6. **High School Aesthetic:**
    - Professional, not cartoonish.
    - Clean typography and spacing.
    - Modern color palette (blues, greens, grays).
    - Minimalist icon usage.
